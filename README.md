# Act

Unity prototype workspace for a multiplayer action game.

The current first slice is an editor toolkit:

- Combat animation preview and gameplay marker authoring.
- Level whitebox placement and blockout JSON export.
- Curated test model and animation source list.

Open the tools from Unity:

- `Tools/Act Toolkit/Combat Animation Editor`
- `Tools/Act Toolkit/Level Blockout Editor`
- `Tools/Act Toolkit/Testing Asset Sources`
